## Willkommens- Website

Eine persönliche Willkommens-Website, die Informationen über mich darstellt.

## Beschreibung
Dieses Projekt dient als Einstieg in die Webentwicklung. Ziel war es, eine übersichtliche Website zu erstellen, die meinen Lebenslauf, Fachhochschulzeugniss, sowie Kontaktmöglichkeiten darstellt.

## Features
- Übersichtliche Darstellung von persönlichen Informationen
- Strukturierter Aufbau mit HTML
- Gestaltung und Layout mit CSS

## Technologien
- HTML
- CSS

## Lernziele
- Grundverständnis für den Aufbau von Webseiten
- Arbeiten mit HTML-Strukturen (z. B. Sections)
- Gestaltung mit CSS (Farben, Layout und Abstände)

## Status
Das Projekt ist größtenteils abgeschlossen, wird weiterhin verbessert.
